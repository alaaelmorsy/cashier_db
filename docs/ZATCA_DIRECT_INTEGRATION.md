# الربط الإلكتروني المباشر مع ZATCA (بدون وسيط)

اعتبارًا من هذا الإصدار يدعم الكاشير وضعين للربط الإلكتروني (المرحلة الثانية):

## الوضعان

| | الوضع القديم (legacy) | الربط المباشر (direct) |
|---|---|---|
| الآلية | وسيط Java/Payara محلي على المنفذ 8080 | توقيع وإرسال من داخل البرنامج مباشرة لبوابة الهيئة (`@talha7k/zatca`) |
| المتطلبات | Java + Payara مثبتان | لا شيء — البرنامج فقط |
| من يستخدمه | العملاء المربوطون قبل هذا الإصدار | الربط الجديد والانتقال الطوعي |

## ضمانة عدم إعادة الربط

- عند أول تشغيل بعد التحديث: أي جهاز كان عليه «تفعيل الربط الإلكتروني (ZATCA)» يُثبَّت تلقائيًا على الوضع القديم ويستمر بالعمل **كما هو بلا أي خطوة إضافية**.
- لا يوجد أي تحويل تلقائي إلى الوضع المباشر إطلاقًا — الانتقال يتم فقط بإكمال المستخدم كل خطوات الربط المباشر حتى شهادة الإنتاج، وبعد تأكيد صريح يوضح بدء سلسلة فواتير جديدة لدى الهيئة.

## التبديل بين الوضعين

التبديل يتم **من شاشة الربط الإلكتروني فقط** (وليس من الإعدادات): شاشة ZATCA → زر «الربط المباشر (بدون وسيط)» → بطاقة «وضع الربط النشط»:
- **التحويل للوضع القديم**: متاح دائمًا، ويُبقي شهادات الربط المباشر محفوظة للعودة لاحقًا.
- **التحويل للربط المباشر**: متاح فقط بعد إكمال خطوات الربط حتى شهادة الإنتاج.

## خطوات الربط المباشر

شاشة ZATCA → **الربط المباشر (بدون وسيط)**:
1. إدخال بيانات المنشأة والعنوان الوطني (تُعبّأ افتراضيًا من إعدادات المنشأة).
2. اختيار البيئة (تجريبية/محاكاة/إنتاج) وتوليد CSR.
3. إدخال OTP من بوابة فاتورة → شهادة الامتثال.
4. تشغيل اختبارات الامتثال الستة (تلقائية).
5. إصدار شهادة الإنتاج → اكتمل الربط.

## ملاحظات تشغيلية

- انقطاع الإنترنت لا يوقف البيع: الفواتير تُطبع وتُحفظ «بانتظار الإرسال». يبدأ فحص الإرسال عند تشغيل البرنامج، ثم يُعاد كل 15 دقيقة للفواتير غير المرسلة وقيد الانتظار والفاشلة، مع احترام تاريخ بداية الإرسال والحفاظ على تسلسل ICV/PIH ومنع الازدواج.
- فاتورة العميل ذي الرقم الضريبي تُرسل كفاتورة ضريبية (مخالصة) وتتطلب إكمال «عنوان الفاتورة الضريبية» في بطاقة العميل (شارع/مبنى/حي/مدينة/رمز بريدي).
- الأسرار (المفاتيح والشهادات) مشفرة AES-256-GCM بمفتاح داخل قاعدة البيانات — نسخة قاعدة البيانات وحدها تكفي للانتقال لجهاز جديد.
- تنبيه قبل انتهاء الشهادة بثلاثين يومًا؛ عند انتهائها يتوقف الإرسال فقط ويستمر البيع.
- إدارة الربط والإرسال متاحة على الجهاز الرئيسي فقط.
